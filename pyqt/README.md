# PyQt5

## Installation

* PyQt5

> $ pip install PyQt5

* QtDesigner

> $ pip install pyqt5-tools

## Usage

pydesigner로 만든 ui file을 python file로 만드는 방법.

> $ pyuic5 -x xxx.ui -o xxx.py
